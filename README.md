# Amazon MediaLive (amazon-medialive)
AWS Elemental MediaLive is a broadcast-grade live video processing service that creates high-quality video streams for delivery to broadcast TVs and internet-connected multiscreen devices. It allows you to create live streams from various sources and deliver them to viewers anywhere.

## APIs

### AWS Elemental MediaLive API
The AWS Elemental MediaLive API provides programmatic access to create and manage channels, inputs, input security groups, multiplexes, and reservations for live video encoding and distribution.
